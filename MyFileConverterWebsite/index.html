About This Project

Student Project - File Format Converter

This file conversion tool was developed as a student project to demonstrate practical software development skills including Python programming, Windows registry integration, and executable packaging.

The application integrates seamlessly with Windows Explorer, providing right-click context menu options for instant file format conversion without opening additional applications.
